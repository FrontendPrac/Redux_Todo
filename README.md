# TodoList 리덕스 리펙토링

> 기능

- 추가 : addTodo
- 삭제 : deleteTodo
- 수정 : switchTodo
- 페이지 이동 : route

- 동적 라우팅 구현

> 적용한 개념

- action, actioncreator, type, payload, reducer
- map, fileter, find
- react-router-dom
- useState, useSelector, useDispatch
- useParams, useNavigate, Link
- styled-components
