# TodoList 리덕스 툴킷 리펙토링

> 기능

- 조회 : __getTodos(12.21)
- 추가 : __addTodo
- 삭제 : __deleteTodo
- 수정 : __switchTodo
- 페이지 이동 : 동적 라우팅(12.19)
- 서버 연결 및 통신(12.21)

- useState, useSelector, useDispatch
- useParams, useNavigate, Link
- styled-components
- redux-toolkit(12.20)
- jason-server(12.20)
- axios, createAsyncThunk(12.21)
- fulfillWithValue, rejectWithValue(12.21)
- extraReducers(12.21)
